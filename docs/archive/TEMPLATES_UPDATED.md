## Copy-Paste Workflow

### For Standard Content:

**From Brainstorming Document:**
1. **Title/Descriptions** → Copy from table cells
2. **Keywords for CMS** → Copy from "All Keywords (comma-separated for CMS)" section
3. **Quotes** → Copy blockquote text

**From Copy Revision Document:**
1. **Revised title** → Copy from "Proposed Revision" column
2. **Keywords** → Copy from "SEO Keywords" section

### For Digital Shorts:

**Each clip has:**
```
**Title:** [text] - _[XX chars]_           ← Copy just the text
**Long Description:** [text] - _[XX chars]_ ← Copy just the text
**Keywords:** [comma-separated list]        ← Copy entire line
**Social Media Tags:** #tag1 #tag2 #tag3   ← Copy entire line
```

No tables to navigate - just select and copy the content after each label.

## Why This Format Works

### For CMS Integration:
- **Keywords in one line** → Paste directly into keyword field
- **No table parsing needed** → Text selection is straightforward
- **Character counts visible** → Verify limits before pasting
- **Multiple options provided** → Easy to compare without format confusion

### For Quality Control:
- **Content summary** → Quick verification of correct transcript
- **Keyword categorization** → Understand SEO strategy
- **Metadata header** → Track program-specific rules applied

### For Workflow Efficiency:
- **Reduced formatting cleanup** → CMS paste is cleaner
- **Clear labeling** → No confusion about what goes where
- **Inline char counts** → Immediate verification
